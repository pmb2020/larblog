怎么把laravel项目上传到GitHub上
前提：先在GitHub上创建好一个仓库
1、初始化：git init
2、添加GitHub上的仓库地址
    git remote --v   查看git远程仓库地址
    发现是空的，很正常，下面我们来添加我们的远程仓库地址

    git remote add origin git@github.com:pmb2020/larblog.git
    再使用上面的命令查询，就会有我们更添加的仓库地址了
3、代码提交到GitHub
